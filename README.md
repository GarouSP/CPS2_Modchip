# CPS2_Modchip
Modchip for suicide free CPS2

Burn on a PIC12F675 the HEX of your romset.

Pinout of the PIC is :

```text
           +---\/---+
     +5V   |1*     8|  GND
     DATA  |2      7|  SETUP2
           |3      6|  CLOCK
           |4      5|  SETUP1
           +--------+
```
